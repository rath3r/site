<h2><span class="editsection">[<a href="/index.php?title=Sites&amp;action=edit&amp;section=2" title="Edit section: bluematrix.com">edit</a>]</span> <span class="mw-headline" id="bluematrix.com">bluematrix.com</span></h2>
<h3><span class="editsection">[<a href="/index.php?title=Sites&amp;action=edit&amp;section=3" title="Edit section: Dev">edit</a>]</span> <span class="mw-headline" id="Dev">Dev</span></h3>
<pre>git clone git@bitbucket.org:livinggroup/bluematrix.git .
</pre>
<p>Follow instructons in the README.txt file. Develop using
</p>
<pre>grunt server
</pre>
<h3><span class="editsection">[<a href="/index.php?title=Sites&amp;action=edit&amp;section=4" title="Edit section: Test">edit</a>]</span> <span class="mw-headline" id="Test">Test</span></h3>
<p>Url: <a rel="nofollow" target="_blank" class="external text" href="http://bluematrix.living-digital.com">bluematrix.living-digital.com</a>
</p><p>Setup:
</p><p>Firstly create new subdomain in plesk.
</p><p>By adding a vhost.conf file we can add custom configuration and especially change the DocumentRoot to a more isolated location rather than the combined location in /var/www/vhosts/living-digital.
</p><p>To do this:
</p>
<ul><li> Create <code>vhost.conf</code> in <code>/var/www/vhosts/bluematrix.living-digital.com/conf</code> and add the following line to it.
</li></ul>
<pre>DocumentRoot /var/www/vhosts/bluematrix.living-digital.com/httpdocs/wwwroot
</pre>
<ul><li> Re-configure the sub-domain
</li></ul>
<pre>/usr/local/psa/admin/sbin/websrvmng --reconfigure-vhost --vhost-name=bluematrix.living-digital.com
</pre>
